## Prerequisites

**Node version 14.x**

### Cloning the repository

```
git clone https://github.com/eugenetye/ts-node-mongo-restAPI.git
```

### Install packages

```
npm i
```

### Setup MongoDB URL

In `src/index.ts`:

```js
const MONGO_URL = ''; // DB URI
```

### Start the app

```shell
npm start
```
